Règles d'Utilisation

Maintenir la cohérence de l'identité visuelle

✓ À Faire

  • Toujours utiliser le fond sombre comme base principale
  • Appliquer les dégradés or sur les titres importants
  • Conserver un fort contraste texte/fond pour la lisibilité
  • Utiliser les violet/teal comme couleurs dominantes
  • Incorporer des éléments naturels dans les visuels (plumes, cristaux, herbes)
  • Garder une atmosphère mystique mais accueillante
  • Utiliser le magenta/fuchsia avec parcimonie comme accent
  • Laisser beaucoup d'espace négatif (respiration)

✗ À Éviter

  • Ne jamais utiliser de fond blanc pur (#FFFFFF)
  • Ne pas mélanger plus de 3 couleurs dans un même bloc
  • Éviter les polices sans-serif génériques (Arial, Helvetica)
  • Ne pas utiliser de couleurs néon ou trop saturées
  • Éviter les images de stock génériques (bougies clichés)
  • Ne pas surcharger avec trop de symboles ésotériques
  • Éviter les backgrounds trop clairs qui cassent l'ambiance
  • Ne pas utiliser de bordures épaisses ou arrondies excessives

Recommandations UI/UX Pro

Intelligence design issue du skill UI/UX Pro Max

Style UI Recommandé

Style Primaire

Dark Mode OLED + Luxury

Fond noir profond (#0A0A12) avec accents lumineux dorés et violets. Contraste WCAG AAA pour une lisibilité parfaite. Effets de lueur subtils (text-shadow, box-shadow) pour l'atmosphère mystique. Optimisé OLED pour économie de batterie sur mobile.

Performance: Excellent | Accessibilité: WCAG AAA | Complexité: Basse

Style Secondaire

Liquid Glass + Glassmorphism

Effets de verre translucide pour les cartes produits et les modales. backdrop-filter: blur(15px) avec bordures subtiles rgba. Crée une sensation de profondeur magique et de luxe premium. Parfait pour les cartes produits et les overlays.

Performance: Modérée | Accessibilité: WCAG AA | Complexité: Moyenne

Bonnes Pratiques UX

Navigation

  • ✓ Nav sticky avec padding-top compensé
  • ✓ Max 6-7 items dans le menu
  • ✓ Logo cliquable vers l'accueil
  • ✓ Menu hamburger sur mobile
  • ✓ CTA visible dans le header

Animations

  • ✓ Transitions 200-300ms maximum
  • ✓ Hover = changement curseur + visuel
  • ✓ Click/tap pour actions primaires
  • ✗ Pas d'animations infinies décoratives
  • ✓ prefers-reduced-motion respecté

E-commerce

  • ✓ Photos produits fond sombre uniforme
  • ✓ Prix en couleur or, bien visible
  • ✓ Bouton "Ajouter au panier" contrasté
  • ✓ Filtres par catégorie clairs
  • ✓ Avis clients avec étoiles dorées
